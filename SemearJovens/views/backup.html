Backup

Faça backup completo do banco ou exporte planilhas por menu.

Backup completo (Banco)

Gera um arquivo .sql com todos os dados do sistema (inclui tabelas e relações).

Esse backup não inclui arquivos físicos de anexos.
Exportar planilhas por menu

Gera um Excel com abas separadas para cada menu de dados.

Menus sem dados exportáveis (ex: Anexos) ficam apenas no backup do banco.
Importar backup (Banco)

Restaura um arquivo .sql completo no banco de dados.

Use somente arquivos .sql gerados pelo backup do sistema.
Importar planilha

Importa dados da aba “Cadastro Jovens” do Excel de backup.

Para restaurar todos os dados, use o backup do banco.
